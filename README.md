# Plants Vs Zombies Unity Learning Project

This is a personal Unity project where I am rebuilding a Plants vs Zombies-style game in my free time to learn game development with the help of agentic AI.

I recently graduated in **Artificial Intelligence and Data Science**. My background is mostly software engineering, AI systems, optimisation research, and web/product development, so this repo is my practical entry point into Unity and game architecture.

## Focus

The project is designed around an **environment plus agents** model:

- Environment: board grid, tiles, lanes, sun economy, waves, win/lose rules.
- Agents: plants, zombies, projectiles, sun pickups, lawn mowers.

The first target is a small playable core level with a 5 x 9 board, Peashooter, Sunflower, Basic Zombie, sun collection, projectile combat, waves, win/lose, and restart.

## Build

```sh
make build
make run
```

`make run` opens the built macOS app from `Builds/PVZ.app`.

## Asset Note

This is a private learning project. If it is ever shared publicly, the assets should be replaced with original or properly licensed art and audio.
